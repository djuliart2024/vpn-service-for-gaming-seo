## FAQ

**1) Will a vpn service for gaming reduce ping?**  
Sometimes. It can improve routing or avoid ISP congestion, but a bad server choice will add latency. Pick the closest server first, then test.

**2) Does this help with DDoS during competitive play?**  
Yes—masking your real IP helps. Pair it with **Kill Switch** so you don’t leak if the tunnel drops.

**3) Is streaming/voice chat affected?**  
It shouldn’t be. If voice chat gets flaky, switch to a nearer server or a less loaded region.

**4) What should I enable first?**  
Enable **Kill Switch**, then connect. After that, choose the nearest region from the **Global Servers** list.
